Acerca de Dragon Ball API
¿Qué es esto?

La API de Dragon Ball es una API REST completa inspirada en la icónica serie de televisión Dragon Ball. Esta API otorga a los usuarios acceso a una extensa base de datos que cuenta con cientos de personajes, imágenes, transformaciones y planetas del universo Dragon Ball. Incluye información canónica derivada de varias series, incluyendo Dragon Ball Z, Dragon Ball GT, Dragon Ball Super, así como películas, y un toque de Dragon Ball Heroes.

¿Quién eres?

Soy Porozo Caicedo Jose, un chico que ama desarrollar cosas. Estudié en Relevant en el Polo de Contenidos Digitales en Málaga, y tengo una profunda pasión por Dragon Ball.

¿Por qué construiste esto?

Mi pasión por Dragon Ball y el deseo de ayudar a mis compañeros estudiantes me llevaron a desarrollar este recurso. Como estudiante, entiendo los desafíos que enfrentamos y quería contribuir con algo valioso para hacer el proceso de aprendizaje más agradable. Este proyecto es mi forma de combinar mi amor por Dragon Ball con un compromiso de apoyo a los estudiantes, proporcionando una herramienta útil. Es una iniciativa personalizada para hacer que aprender y explorar el universo de Dragon Ball sea un poco más emocionante y educativo para todos los involucrados.
